# ComfyUI_kkTranslator_nodes

这些 nodes 主要用于把提示词从其他语言翻译成英文.

## 安装
1. 克隆这个仓库到 ComfyUI 的 custom_nodes 文件夹中。
```
cd custom_nodes
git clone https://github.com/chaojie/ComfyUI-DragNUWA
```
2. 运行以下命令安装依赖
```
python -m pip install -r requirements.txt
```
3. 重启你的 ComfyUI
